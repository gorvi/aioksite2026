# 数据库连接工具脚本说明

## 🔧 保留的实用脚本

### 1. `auto_tunnel.sh` - 自动建立SSH隧道 ⭐ **推荐使用**
```bash
./auto_tunnel.sh
```
- **功能**: 自动建立SSH隧道连接到远程数据库
- **端口**: 本地3307 → 远程3306
- **优点**: 全自动，无需手动输入密码
- **状态**: 后台运行，不占用终端

### 2. `test_tunnel.sh` - 检查隧道状态
```bash
./test_tunnel.sh
```
- **功能**: 检查SSH隧道是否正在运行
- **显示**: 端口状态、进程信息、连接测试
- **用途**: 故障排查和状态确认

### 3. `start_db_tunnel.sh` - 原始隧道脚本
```bash
./start_db_tunnel.sh
```
- **功能**: 交互式建立SSH隧道
- **端口**: 本地3306 → 远程3306
- **注意**: 需要手动输入密码，会占用终端

## 📋 使用建议

### 日常使用
1. **启动隧道**: `./auto_tunnel.sh`
2. **检查状态**: `./test_tunnel.sh`
3. **MCP配置**: 端口3307已配置

### 故障排查
- 如果数据库连接失败，先运行 `./test_tunnel.sh` 检查隧道状态
- 如果隧道断开，重新运行 `./auto_tunnel.sh`
- MCP MySQL工具会自动使用端口3307连接

### 重要提醒
- SSH隧道在后台运行，重启电脑后需要重新建立
- 密码已内置在脚本中，确保脚本文件安全
- 如果更改服务器密码，需要更新 `auto_tunnel.sh` 脚本
