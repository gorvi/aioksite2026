> **中等**

## 五、理解与安慰型说明（非常重要）

**说明文案**

> 这些表现并不代表能力不足或不够努力。
> 许多人在压力大、任务多或长期疲劳的状态下，都会出现类似情况。

## 六、建议区（非医疗）

**标题**

> 给你的参考建议

**建议示例（根据维度动态显示）**

* 尝试将任务拆分为更小、更明确的步骤
* 使用提醒、清单等外部工具减轻记忆负担
* 在专注任务前，为自己预留一个“启动缓冲时间”

（文案原则：**具体、温和、不带治疗色彩**）

## 十、UI 文案风格统一原则（给设计师）

* 不使用：**障碍 / 疾病 / 严重 / 病理**

* 所有结论避免“你就是 / 你存在”
