# SCL-90 常模数据获取指南

## 📚 主要获取渠道

### 1. **学术数据库和论文**

#### 中文数据库
- **中国知网（CNKI）**
  - 网址：https://www.cnki.net
  - 搜索关键词：`SCL-90 常模`、`症状自评量表 常模`、`SCL-90 中国人群`
  - 推荐论文：
    - 《症状自评量表(SCL-90)河南省青少年区域性常模的建立和心理测量特性验证》
    - 《中学生SCL-90信度、效度检验与常模建立》

- **万方数据**
  - 网址：https://www.wanfangdata.com.cn
  - 搜索关键词同上

- **维普资讯**
  - 网址：https://www.cqvip.com

#### 英文数据库
- **PubMed**
  - 网址：https://pubmed.ncbi.nlm.nih.gov
  - 搜索：`SCL-90 norm data Chinese population`

- **Google Scholar**
  - 网址：https://scholar.google.com
  - 搜索：`SCL-90 Chinese norms mean standard deviation`

### 2. **专业机构知识库**

#### 中国科学院心理研究所
- **机构知识库**：https://ir.psych.ac.cn
- 搜索：`SCL-90` 或 `症状自评量表`
- 可能包含：
  - 中学生群体的常模数据
  - 信度、效度检验报告
  - 不同人群的常模建立研究

#### 其他心理学研究机构
- 各大学心理学系的研究成果
- 心理健康研究中心的数据

### 3. **专业心理测评网站和资源**

#### 国内资源
- **修水心理协会**（公众号/网站）
  - 提供 SCL-90 使用说明和常模数据参考

- **心理测评专业网站**
  - 部分网站提供常模数据参考
  - 注意：需要验证数据的权威性

#### 国际资源
- **Mental Health America**
- **Psychological Assessment Resources (PAR)**

### 4. **专业书籍和手册**

#### 推荐书籍
- 《心理卫生评定量表手册》
- 《常用心理评估量表手册》
- 《心理测量学》相关教材

这些书籍通常包含：
- SCL-90 的标准化常模数据
- 不同人群的均值、标准差
- 分级标准（轻/中/重/极重）

### 5. **开源数据集**

#### GitHub / 学术数据平台
- 搜索：`SCL-90 dataset`、`psychological assessment norms`
- 注意：需要验证数据的合法性和准确性

## 📊 常模数据通常包含的内容

### 1. **总体常模**
- 总均分：均值 ± 标准差
- 原始总分：均值 ± 标准差
- 阳性项目数：均值 ± 标准差

### 2. **各维度常模**
每个维度（9个因子）的：
- 原始分：均值 ± 标准差
- 均分：均值 ± 标准差
- 分级标准（轻/中/重/极重）

### 3. **不同人群常模**
- 按年龄分组（青少年/成人/老年）
- 按性别分组
- 按地区分组（如：河南省、全国）

## ⚠️ 使用注意事项

### 1. **数据适用性**
- 确认常模数据是否适用于您的目标人群
- 考虑地区、文化、年龄等因素的差异

### 2. **数据来源权威性**
- 优先使用学术机构发布的数据
- 验证数据的样本量和代表性
- 确认数据的发布时间（常模可能随时间变化）

### 3. **合规性**
- 确保数据使用符合相关法规
- 注意数据版权和使用许可
- 如果用于商业用途，需要获得相应授权

### 4. **数据格式**
常模数据通常以以下格式呈现：
```typescript
{
  dimension: 'somatization',
  mean: 1.37,
  stdDev: 0.48,
  sampleSize: 1388,
  population: 'Chinese adults',
  reference: '论文/书籍来源'
}
```

## 🎯 推荐获取路径

### 快速获取（适合开发测试）
1. 搜索中国知网相关论文
2. 查找《心理卫生评定量表手册》等专业书籍
3. 参考专业心理测评网站的公开数据

### 正式使用（需要权威数据）
1. 联系专业心理测量机构
2. 购买正版常模数据
3. 或与学术机构合作获取授权

## 📝 数据引用格式

如果使用常模数据，建议在应用中注明：
- 数据来源（论文/书籍/机构）
- 适用人群
- 数据获取时间
- 免责声明

## 💡 建议

对于收费应用（2元测试），建议：
1. **短期方案**：使用公开的学术论文中的常模数据（需注明来源）
2. **长期方案**：联系专业机构获取正版授权，或与学术机构合作

这样可以：
- 保证数据的权威性
- 避免版权风险
- 提升应用的专业性
